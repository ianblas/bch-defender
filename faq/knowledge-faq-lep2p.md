# Knowledge FAQ (LEP2P Expansion)

## What is an xPub?

An xPub is an extended public key that can be used to derive receiving addresses without exposing spending control.

That makes it useful in some merchant and POS flows.

## Why do alias systems matter in BCH?

Alias systems can reduce friction for users by making payment identifiers easier to use than raw addresses alone.

They matter because usability can help adoption.

## What is OP_RETURN useful for?

At a high level, OP_RETURN can be used for lightweight on-chain data anchoring and record-related use cases.

A common pattern is anchoring hashes or small pieces of data rather than treating the chain like a raw file dump.

## Why does network activity matter?

Because useful activity reflects utility.

A money system is not judged only by price. Real usage, fee generation, and useful applications matter too.

## Why do miners matter in BCH?

Miners matter because they help secure the network and follow economic incentives.

A good beginner explanation should connect miners to network security and incentives rather than hero worship.

## Why would a BCH user care about Tor or I2P?

Because network privacy and censorship resistance can matter when a user wants more resilient access to tools, wallets, and communications.

## What is CashFusion for?

CashFusion is a privacy tool. A good explanation should treat it as part of privacy practice and optionality, not as magic invisibility.

## Can BCH be useful for notary-style timestamping?

Yes, in lightweight ways.

A common use is anchoring a hash or proof of existence on-chain rather than storing full raw records directly in every transaction.
