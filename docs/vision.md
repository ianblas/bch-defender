# BCH Defender Vision

BCH Defender exists to help teams communicate, support, and expand real Bitcoin Cash usage.

In addition to argument and advocacy bots, this repository supports practical workflows for:

- onboarding bots for new BCH users
- merchant support bots for BCH acceptance
- community support workflows inspired by real BCH grassroots ecosystems

The goal is practical adoption: helping normal users and businesses use BCH as peer-to-peer electronic cash.
