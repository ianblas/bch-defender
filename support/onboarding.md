# Onboarding Paths

## New BCH users

1. Install a reputable BCH wallet.
2. Back up seed phrase securely.
3. Receive a small test amount.
4. Send a small payment to learn the flow.
5. Learn basic safety and confirmation practices.

## New BCH merchants

1. Choose acceptance method (wallet, processor, plugin).
2. Run test transactions before launch.
3. Train staff on QR flow and confirmation policy.
4. Set refund and accounting procedures.
5. Announce BCH acceptance to customers.

## First-week checklist

- Complete one inbound and one outbound test payment.
- Confirm backups are in place.
- Document support contacts and incident steps.
