# User Support Guide

## Sending BCH

1. Confirm the recipient address and network (BCH).
2. Enter amount.
3. Review fee and details.
4. Send and save transaction ID when needed.

## Receiving BCH

1. Generate a receive address in your wallet.
2. Share address or QR code.
3. Wait for broadcast and confirmations based on risk level.

## Address checking basics

- Copy-paste carefully; avoid manual typing.
- Verify first and last characters.
- Confirm address format is BCH-compatible.

## Confirmations

- Small in-person payments may be accepted quickly by merchant policy.
- Larger payments should wait for one or more confirmations.
- Risk tolerance determines confirmation policy.

## Common wallet mistakes

- Sending to wrong or incompatible network
- Losing seed phrase backup
- Trusting fake support contacts
- Reusing old addresses when privacy matters

## Basic safety

- Back up seed phrase offline
- Enable device security and wallet PIN/biometric lock
- Keep software updated
- Never share seed phrase or private keys
