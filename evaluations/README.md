# Evaluations

## Purpose

This directory defines how BCH Defender responses should be judged.

Datasets provide the cases.

Evaluations provide the criteria for deciding whether a response is good, bad, safe, useful, and BCH-aligned.

## Evaluation Priorities

Across categories, evaluations should reward responses that are:

- accurate
- operationally useful when needed
- BCH-first in framing
- calm and clear
- explicit about uncertainty when needed
- free of avoidable dangerous advice
